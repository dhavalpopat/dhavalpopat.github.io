Some of my projects...

Event Finder – Search Nearby Events
Backend Development, Summer 2018

Event Finder is a web application that displays events happening in the local neighborhood by populating the map using D3.js based on user’s preferences. Whenever user sends login request, an authentication token gets generated along with the payload, and is saved in local storage at client side. Furthermore, I implemented a middleware function to authorize user request using the authentication token and private key before forwarding it to route handler. The local events data is fetched by hitting an external API using basic auth. I also built an API endpoint to get information of users and restricted its access by verifying user roles with admin privileges.

Mask.It – Overlay Masks on Facial Images
Computer Vision, Spring 2018

The goal of this project is to apply various masks on facial images using landmark detection and mask morphing. The facial landmarks are retrieved by segmenting skin region from the prominent facial features and analyzing its frequency projections. Further, I morphed various masks using non-linear alignment of images with warping to match relevant landmark points and fit the face. The resultant masks are overlaid onto facial image using Lanczos interpolation and alpha blending giving an effect of augmented reality.
